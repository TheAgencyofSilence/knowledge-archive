# Этап 3. Создание

<h2>Компоненты</h2> 

Ваш предмет не создается на пустом место, для этого всегда нужны некоторые компоненты – различные материалы для изготовления вашей предмета. Компоненты также бывают четырех типов:

•  **Технические** 

•  **Химические** 

•  **Творческие** 

•  **Оккультные** 

Разумеется, для создания предмета из определенной сферы нужны такие же определенные компоненты. Например, создавая предмет **Технической сферы**, вам необходимы именно **технические** компоненты.

Есть несколько основных способов получить компоненты: 

 •  Вам может посчастливится отыскать их на игровых партиях 

 •  Ваш персонаж может получать их, как награду за уникальные Инициативы.

 •  Вы можете выменять их у других персонажей за деньги/предметы/услуги на  <a href="https://t.me/+BR7SoYrBlqo0NjM6">доске Купи/Продай</a>

Для того чтобы начать создания предмета, вам необходимо собрать или купить компоненты 6 КТ в количестве У.Е. будущей вещи и найти 4 соответствующих компонента по уровню КТ  (так называемый «Старт»). Добыв эти компоненты, вы можете приступать непосредственно к процессу созданию вашего предмета.

<h2>Процесс создания</h2> 

Чем выше сила и функционал создаваемого предмета – тем сложнее его создать. Чем выше Категория Технопакта – тем больше усилий необходимо приложить, создавая вещь. Более того, сам процесс создания сложных и редких вещей, своего рода шедевров, отличается:

 •  Предметы КТ 5-6 - это базовые, почти доступные всем в мире Столицы изделия. Для их изготовления все ещё нужен чертеж, набор компонентов и проверка крафта по схеме ниже, результаты которой нужно занести в <a href="https://t.me/+Jzg8nbtpqxUyZjQ6">Крафт</a> . 

 •  Изобретения КТ 3-4 - другое дело, это действительно уникальные авторские изобретения. Чтобы их создать нужны не только чертеж и компоненты, но и более сложная проверка, и дополнительный процесс - лицензирование. Кроме прочего, создание Изобретения 3КТ можно создать только единожды, после создания чертеж больше нельзя использовать.

Вещь всегда несет в себе след мастера его создавшего, что отражается в названии: Тихий револьвер Уайта, Инкубатор Хеймса-Грабова. 

Для создания Изобретения нужно написать в <a href="https://t.me/+Jzg8nbtpqxUyZjQ6">Крафт</a> для оценки, и <a href="https://t.me/+SHHh4Q65c-liNjQy">Часовым</a> они помогут сформулировать подходящую Инициативу по созданию, которая будет включать в себя и проверку крафта, и получение лицензии. **Название инициативы - название будущего  Изобретения, включающее имя крафтера.**
