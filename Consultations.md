# Консультации

Консультация это сторонняя помощь крафтеру с сборкой вещи по чертежу.

<table class="simple-table">

  <tr>
    <td class="left"> <strong> Консультация в родной сфере крафта может:</strong></td>
  </tr>

  <tr>
    <td class="left">  • Дополнительно опустить броню на 1 категорию.
</td>
  </tr>

  <tr>
    <td class="left"> • Добавить доп 2 кубика на соответствующий бросок при крафте ( по правилу создание преимущества).</td>
  </tr>

  <tr>
    <td class="left"> • Расширить предел успехов на 2.</td>
  </tr>

</table>


Бонус работает только на <u>один бросок за процесс создания.</u>

**Консультация в сфере, которая не является квалификацией персонажа:**

Дает возможность, создать **Предмет** без нужных знаний в этой области. 

<div class="note note-gray">
  <p><strong>Пример:</strong></p><br>
  <p>Снайперская винтовка Застава Специального калибра 3 может стрелять патроном со свойством Разрывной 3</p><br>
  <p>Модельер хочет покрыть вещи спец раствором, чтобы дурманить людей. Но он, не Химик и никогда не был  им. </p><br>
  <p>Этим персонажам, нужна консультация специалиста для того чтобы создать предмет в сфере не их профиля. После использование ваучера, игрок может создавать предмет одним навыком. (а не несколькими).</p>
</div>

Такая консультация <u>может быть выдана исключительно, как награда за партию. </u>(Слова других агентов не работают).

**Источники консультаций:**

 •  Задействование Слова Агента с подходящим навыком с числом успехов не меньше чем УЕ Предмета / Изобретения.

 •  Награда за партию.


**Ограничение:**

 •  Один ассистент может предоставить консультацию единожды за бросок крафта, только на один навык. Далее этот ассистент не может дать бонус еще раз (даже при наличии еще одной консультации).

 •  Задействование консультации необходимо заявить до броска.
